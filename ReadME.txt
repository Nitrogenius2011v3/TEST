MOD_LASTEST.cap - Это модифицированная базовая система ввода-вывода для материнской платы ASUS Z170-K.
В данной неофицальной версии предусмотрена поддержка модифицированных процессоров CoffeLake (8-9 Gen), удалена ошибка, 
связанная с невозможностью обновления BIOS через MS.DOS, максимально увеличен список поддерживаемых (не модифицированных) процессоров
!!!ВНИМАНИЕ!!! Установка осуществляется только через программатор CH341A, EZP2019, Postal.
!!!ВНИМАНИЕ!!! При первом включении установить значения ic load line и dc load line на 0,01 и включить HyperThreading
